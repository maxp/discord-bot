# Deployment

Этот документ должен стать runbook для запуска бота вне локальной машины.

## Что сюда добавить после появления runtime

- где крутится бот;
- как передаются secrets;
- как выполняется запуск и перезапуск;
- как смотреть логи;
- как обновлять приложение без ручных сюрпризов;
- какие health checks и smoke checks использовать после деплоя.

## Минимальный операционный чеклист

- bot token и прочие secrets не лежат в репозитории;
- production intents и install settings совпадают с ожидаемым поведением;
- после запуска бот виден online и отвечает на базовую команду;
- есть понятный способ проверить reconnect и startup failures.
