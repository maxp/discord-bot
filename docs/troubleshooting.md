# Troubleshooting

Сюда нужно складывать не теорию, а реальные способы разбирать типовые сбои.

## Хорошие кандидаты для этого файла

- бот не подключается к Gateway;
- interaction истекает по времени;
- бот получает `Missing Access` или `Missing Permissions`;
- ответы не доходят или приходят с задержкой;
- reconnect loop;
- rate-limit проблемы;
- неверный token или сломанная конфигурация окружения.

## Текущие проверки

### Бот не стартует из-за token

Возможная причина: `DISCORD_BOT_TOKEN` пустой или отсутствует в `.env`.

Как проверить:

- убедиться, что `.env` существует;
- проверить, что `make run` загружает `.env`;
- ошибка из `discord-bot.discord.jda/require-bot-token!` будет иметь сообщение `discord-bot-token is empty`.

### Прокси не применяется или задан неверно

Возможная причина: `DISCORD_PROXY_URL` пустой, имеет неподдерживаемую схему или не содержит host.

Как проверить:

- поддерживаются только `http` и `https`;
- parsing выполняет `discord-bot.discord.proxy/parse-proxy-url`;
- для URL без port используются default ports `80` для `http` и `443` для `https`.
