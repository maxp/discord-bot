# Troubleshooting

Сюда нужно складывать не теорию, а реальные способы разбирать типовые сбои.

## Хорошие кандидаты для этого файла

- бот не подключается к Gateway;
- slash-команды не появляются;
- interaction истекает по времени;
- бот получает `Missing Access` или `Missing Permissions`;
- ответы не доходят или приходят с задержкой;
- reconnect loop;
- rate-limit проблемы;
- неверный token или сломанная конфигурация окружения.
