# Commands

Этот документ описывает пользовательские команды бота.

## Для каждой команды стоит фиксировать

- имя команды;
- назначение;
- аргументы и их shape;
- где команда доступна: DM-only или иные contexts;
- кто может использовать команду;
- ожидаемый ответ;
- ошибки и ограничения;
- примеры вызова.

## Рекомендуемый шаблон

### `/example`

- Назначение: кратко описать поведение.
- Аргументы: перечислить параметры.
- Контекст: например, только DM.
- Доступ: кто может вызывать.
- Ответ: что получает пользователь.
- Ошибки: типовые ошибки и ограничения.

## Зачем это нужно

Этот файл должен быть полезен одновременно:

- разработчику, который добавляет новую команду;
- тестировщику, который проверяет поведение;
- автору README или пользовательской справки.
